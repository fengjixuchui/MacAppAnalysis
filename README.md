# 课程简介

本课程展示了基于macOS平台的软件分析的原理、步骤以及详细过程。从macOS系统的目录结构的介绍开始，再到具体应用软件结构的介绍，直到简单软件代码的编写，向学员展示了macOS平台软件的基础知识，逻辑和概念，为学员进一步掌握基于苹果平台软件开发和软件安全分析奠定良好的基础。
